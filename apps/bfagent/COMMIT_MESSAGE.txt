- slide_number (IntegerField)

- Indexes: [presentation, slide_number], [presentation, translation_method]
- Unique constraint: [presentation, text_id]

🎯 NEW VIEWS & ENDPOINTS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. start_translation_pipeline (POST)
   - /medtrans/presentations/<id>/translate/
   - Orchestrates 3-step pipeline
   - HTMX-compatible JSON responses

2. presentation_delete (POST)
   - /medtrans/presentations/<id>/delete/
   - Deletes PPTX file & DB entries
   - Available in all statuses (including hanging)

3. presentation_reset (POST)
   - /medtrans/presentations/<id>/reset/
   - Resets to 'uploaded' status
   - Clears all translations

4. presentation_edit (GET)
   - /medtrans/presentations/<id>/edit/
   - Professional edit interface
   - Slide-by-slide text editing

5. presentation_update_texts (POST)
   - /medtrans/presentations/<id>/update/
   - Saves edited translations
   - Sets manually_edited flag

6. presentation_regenerate (POST)
   - /medtrans/presentations/<id>/regenerate/
   - Creates new PPTX with current translations
   - Timestamped filename
