BF Agent - Universal Workflow Orchestration¶

BF Agent ist eine Django-basierte Universal Workflow Orchestration Platform mit Zero-Hardcoding-Philosophie. Alle Konfigurationen sind datenbankgesteuert und über Django Admin verwaltbar.

Bemerkung

Diese Dokumentation wird automatisch aus dem Quellcode generiert (Single Source of Truth).

Schnellstart¶

# Installation
pip install -r requirements.txt

# Django Setup
python manage.py migrate
python manage.py createsuperuser

# Server starten
python manage.py runserver

Features¶

🔧 Handler Framework

38+ spezialisierte Handler für verschiedene Aufgaben. Drei-Phasen-Verarbeitung: Input → Processing → Output.

Handler API-Referenz
🌐 Multi-Domain

Unterstützung für Bücher, Comics, CAD-Analyse, ExSchutz und weitere Domains.

Domains Übersicht
🤖 AI Integration

Multi-Provider-Support: OpenAI, Claude, Ollama. 63% AI-Integration über alle Handler.

AI Integration
📊 Zero-Hardcoding

Datenbankgesteuerte Konfiguration. Runtime-Änderungen ohne Deployment.
