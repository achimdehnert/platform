## 🚀 Key Features

### 1. Consistency Framework (v3.0)
**Location:** `scripts/consistency_framework.py`

**Generates:**
- ✅ Form Mixins (with smart field detection)
- ✅ Class-Based Views (List, Create, Edit, Delete, Detail)
- ⚠️ Templates (partial - being enhanced)
- ⚠️ URLs (shows instructions only)
- ⚠️ Tests (exists but not tested)

**Commands:**
```bash
# Analyze model
python scripts/consistency_framework.py analyze ModelName

# Generate all components
python scripts/consistency_framework.py generate ModelName --force

# Generate specific
python scripts/consistency_framework.py generate ModelName --components views forms
```

### 2. Model Architecture

**Base Models** (`apps/core/models.py`):
- `TimestampedModel` - created_at, updated_at
- `AutoTrackedModel` - total_requests, last_request_at

**Main Models** (`apps/bfagent/models.py`):
- **Agents** - AI agent configurations
- **Llms** - LLM provider configurations  
- **Projects** - Book/content projects
- **Chapters** - Project chapters
- **Worlds** - World/universe definitions

### 3. CRUD Pattern

**Mixin Pattern** (Generated → Manual):
```python
# 1. Generated Mixin (AUTO)
class LlmsFormFieldsMixin:
    class Meta:
        model = Llms
        fields = ['name', 'provider', ...]  # Smart detection

# 2. Manual Form (SAFE TO EDIT)
class LlmsForm(LlmsFormFieldsMixin, forms.ModelForm):
    # Add custom validation
    def clean_name(self):
        ...
```

### 4. Dynamic CRUD System
**Location:** `apps/bfagent/views/crud_views.py`

Alternative to generated views - dynamic routing based on model name.

## 🚨 Critical Rules

### RULE #1: Fix Tools First (MANDATORY)
**NEVER fix generated code manually. ALWAYS fix the generator.**

**Workflow:**
1. See error in generated file → STOP!
2. Find generator (e.g., ViewGenerator Line 584)
3. Fix generator
4. Regenerate
5. Test

**Documented in:**
- `.windsurf/.windsurfrules` (Lines 1-40)
- `memory-bank/@ALWAYS_READ/@RULE_1_FIX_TOOLS_FIRST.md`

### RULE #2: Generator-First Development
All CRUD components MUST be generated by consistency_framework.py

### RULE #3: Relative Imports
```python
from ..models import Llms  # ✅ Correct
from bfagent.models import Llms  # ❌ Wrong
```

### RULE #4: UTF-8 Everywhere
Framework configured for UTF-8 on Windows (Line 8-24)

## 🔧 Common Tasks

### Create New Model CRUD
```bash
# 1. Define model with CRUDConfig
# 2. Migrate
python manage.py makemigrations
python manage.py migrate

# 3. Generate
python scripts/consistency_framework.py generate ModelName --force

# 4. Add URLs manually
# 5. Test
```
