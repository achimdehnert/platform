# @project-context.md - BF Agent: Django Multi-Agent Development Platform

## 🎯 Project Overview

**Project Name**: BF Agent (BookFactory Agent System)
**Project Type**: Django-based Multi-Agent CRUD Platform
**Domain**: AI-powered content creation with intelligent agent management
**Start Date**: 2024-Q4
**Current Phase**: Phase 2 - Consistency Framework & CRUD Automation
**Implementation Strategy**: Tool-first development, generator-based code creation, sustainable architecture

## 👥 Target Users

### Primary Users

- **Authors & Writers**: Multi-genre book authors (Crime, Romance, Sci-Fi, Fantasy, Non-Fiction)
- **Content Creators**: Long-form content developers across genres
- **Independent Publishers**: Small-scale book production workflows
- **Writing Enthusiasts**: Hobbyist writers exploring different genres

## 🎯 Core Value Proposition

"AI-powered book development that combines AutoGen's conversational agents, L2MAC's unlimited text generation, and CrewAI's workflow orchestration for complete book creation."

## 🚀 Key Features (Current Implementation)

### ✅ Completed Features

1. **Agent Management System**
   - CRUD operations with st.data_editor
   - Interactive agent configuration
   - LLM assignment and testing
   - Navigation system optimization

2. **Core Infrastructure**
   - Streamlit-based UI (app/main.py)
   - Database integration (SQLAlchemy)
   - Modular architecture (develop/stable structure)
   - Rule enforcement system

3. **Development Workflow**
   - Folder structure optimization
   - Automatic rule loading
   - Session initialization protocol
   - Focus discipline system

### 🏗️ In Progress

1. **Enhanced Book Automation**
   - Multi-agent workflow orchestration
   - Character agent integration
   - Plot development automation
   - Content generation pipeline

### 📋 Future Features

1. **Advanced AI Integration**
   - Multi-project management
   - Cross-genre template system
   - Advanced analytics and reporting
   - Publication workflow automation

## 🏗️ Technical Architecture

### Current Structure

```text
bookfactory/
├── .windsurf/           # Windsurf configuration & rules
├── memory-bank/         # Optimized memory system
├── develop/             # Development branch
│   ├── app/            # Streamlit application
│   ├── api/            # API services
│   └── agents/         # Agent implementations
├── stable/             # Stable branch
└── README.md           # Documentation
```

### Technology Stack

- **AI Frameworks**: AutoGen, L2MAC, CrewAI
- **Web Interface**: Streamlit + Windsurf
- **Memory System**: File-based + Memory Bank
- **Language**: Python 3.11+
- **Development**: Windsurf IDE integration
