# Quick Start: Super Macro Button Setup

## ⚡ 1 Button in 1 Minute

### **Beispiel: Quick Task Button**

1. **Öffne StreamDeck**

2. **Ziehe "Super Macro" auf leeren Button**
   - Suche in Actions: "Super Macro"
   - Drag auf Button

3. **Konfiguriere:**
   ```
   Titel: Quick Task
   ```

4. **Aktiviere "Load macros from files":**
   - Checkbox ✅

5. **Wähle File:**
   - Klick auf "Choose file..."
   - Navigate zu: `C:\Users\achim\github\bfagent\streamdeck\macros\`
   - Wähle: `02_quick_task.txt`

6. **Speichern** (einfach woanders klicken)

7. **Testen:**
   - Button drücken
   - Öffne Notepad
   - CTRL+V
   - Template sollte erscheinen! ✅

---

## 🔥 Pro Tip: Batch Setup

**Alle Buttons auf einmal:**

1. Erstelle ersten Button komplett
2. **Kopiere** diesen Button (CTRL+C auf Button)
3. **Paste** auf alle anderen Positionen
4. **Ändere nur:**
   - Titel
   - File-Pfad
   - Farbe

**Zeit:** 5 Minuten statt 15! ⚡
