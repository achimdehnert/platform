# StreamDeck Super Macro Files

**Ordner:** Macro-Text-Dateien für alle CASCADE Templates

## 🎯 Verwendung

### **Methode 1: File-Based (Empfohlen)**

1. **Super Macro Button konfigurieren:**
   - Aktiviere "Load macros from files" ✅
   - Short-Press File: Wähle entsprechende `.txt` Datei
   - Speichern

2. **Button drücken:**
   - Template wird automatisch zu Clipboard kopiert
   - CTRL+V zum Einfügen

### **Methode 2: Direct Paste**

1. **Super Macro Button konfigurieren:**
   - Deaktiviere "Load macros from files"
   - Öffne `.txt` Datei in Notepad
   - Kopiere Inhalt
   - Paste in "Short-Press Macro" Feld
   - Speichern

2. **Button drücken:**
   - Template wird zu Clipboard kopiert
