## Übersicht

BF Agent ist ein Django-basiertes System für AI-gestütztes kreatives Schreiben.

### Hauptkomponenten

- **Writing Hub** - Bücher, Kapitel, Charaktere
- **Workflow System** - Automatisierte Content-Generierung
- **Handler-Agent Pattern** - Saubere Trennung von DB und AI-Logik
- **n8n Integration** - Visual Workflow Automation
