Überblick
---------

Das DDL Governance System implementiert einen strukturierten Prozess für:

* **Business Cases** - Fachliche Anforderungen und Problemstellungen
* **Use Cases** - Detaillierte Benutzerinteraktionen
* **ADRs** - Architecture Decision Records
* **Reviews** - Freigabe-Workflow
