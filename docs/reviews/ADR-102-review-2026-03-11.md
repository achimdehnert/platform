## [SUGGEST] S1 — SSL Mode "Full" statt "Full (Strict)"

**Datei:** `ADR-102:214` — `SSL Mode: Full`
**Begründung:** "Full" validiert das Origin-Zertifikat nicht. "Full (Strict)" validiert
gegen Cloudflare Origin CA oder gültige CA. Da Let's Encrypt weiterhin läuft, wäre
"Full (Strict)" die sicherere Wahl — verhindert MITM zwischen CF und Origin.

---

## [SUGGEST] S2 — Port 80/443 extern schließen (P2 seit 2026-03-06)

**Datei:** `ADR-102:126` — Port-Closure als P2 ausstehend
**Begründung:** Seit 5 Tagen offen. Server-IP ist zwar nicht mehr via DNS erreichbar,
aber wer die IP kennt (z.B. aus alten Logs, Shodan) kann den Server direkt erreichen.
`iptables`-Regeln oder `ufw allow from 173.245.48.0/20` (Cloudflare IPs) wäre schnell
umsetzbar. Docker-Networking-Kompatibilität muss getestet werden.
